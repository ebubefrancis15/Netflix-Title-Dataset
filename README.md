# Netflix Data Analysis

## Overview
This project explores the Netflix Titles dataset using Python to identify trends in Netflix's content library.

## Objectives
- Clean the dataset
- Perform exploratory data analysis
- Visualize trends
- Generate insights

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Findings
- Movies dominate Netflix's catalog.
- The United States contributes the highest number of titles.
- TV-MA is the most common rating.
- Most movies are between 91 and 120 minutes long.
